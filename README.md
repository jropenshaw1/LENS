# LENS

**Passive prompt capture and pattern analysis system for AI interactions.**

LENS is a personal diagnostic tool that fires when an AI asks a clarifying question — a signal that the prompt was underspecified. It captures the original prompt, an AI-authored optimized version, and metadata explaining what was missing and why. Over time, this builds a personal dataset that reveals prompt patterns and drives measurable improvement.

## Core Principle

LENS fires when the user is the human author of the prompt. It does not fire in AI-to-AI workflows or coding execution environments. This authorship-based rule governs all scope decisions.

## Governing Philosophy

"When you know better, you do better."
"If you can't measure it, you can't improve it."
"Change is inevitable... growth is optional."

LENS is a self-modifying system. Measuring prompt behavior changes prompt behavior (Heisenberg effect). The system is designed to evolve — not optimized for a static baseline.
